# SQL
Poniższa baza zawiera dane dotyczące pracy ośmiorga pracowników fikcyjnego biura rzeczy znalezionych roznoszących ulotki. Przedstawiona została ona w postaci nieznormalizowanej oraz w pierwszej, drugiej i trzeciej postaci normalnej. Projekt podsumowuje semestralny wstęp do SQLa, jaki realizowaliśmy w ramach przedmiotu Bazy danych.
